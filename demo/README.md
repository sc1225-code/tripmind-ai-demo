# Demo 迭代目录

本目录用于存放 TripMind AI 的各版本可交互 Demo。

## 当前版本

### `mobile-v2`

当前主推版本。

特点：

- 移动端 App 形态
- 首页使用主文案：“向着阳光出发，让每份游学灵感都变成值得期待的旅程。”
- 辅助文案：“整理攻略、记住偏好、对比方案，陪你找到更适合的下一站。”
- 首页加入纸飞机和背包客在路上的视觉
- 点击“出发”进入 AI 问答主入口
- AI 问答采用聊天首页结构，底部常驻输入框
- 输入框上方提供“行程规划 / Plan B / 知识库”分析模式
- 左侧菜单提供 AI 问答和知识库管理入口
- 知识库管理页采用“资料源 + AI 记忆分组”结构

本地预览：

```bash
python3 -m http.server 5175 --directory demo/mobile-v2
```

访问：

```text
http://localhost:5175/
```

## 历史版本

### `mobile-v1`

第一版移动端 Demo。

特点：

- 首页为较抽象的旅行卡片视觉
- 已包含“出发”入口
- 已包含底部四模块导航

本地预览：

```bash
python3 -m http.server 5174 --directory demo/mobile-v1
```

访问：

```text
http://localhost:5174/
```

## 版本维护规则

- 大的流程或视觉调整，新建版本目录。
- 小的文案或样式微调，可以直接在当前主推版本中修改。
- 不删除旧版本，除非明确进入项目清理阶段。
