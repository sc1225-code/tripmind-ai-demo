# TripMind AI Demo

This folder is ready for GitHub Pages deployment.

## Deploy with GitHub Pages

1. Create a new GitHub repository, for example `tripmind-ai-demo`.
2. Upload all files in this folder to the repository root.
3. Open `Settings -> Pages`.
4. Select `Deploy from a branch`.
5. Select `main` and `/root`.
6. Visit the generated GitHub Pages URL.

The demo is a static AI Coding prototype built with HTML, CSS, and JavaScript.
